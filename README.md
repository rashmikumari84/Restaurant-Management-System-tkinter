# 🍽️ Restaurant Management System (Tkinter)

A simple and interactive Restaurant Management System built using Python and Tkinter GUI. This application helps manage customer orders, calculate bills, and display item prices efficiently.

---

## 📌 Features

- Order management for multiple food items
- Automatic cost calculation
- Tax and service charge calculation
- Built-in calculator
- Price list window
- Reset functionality
- Exit button
- Real-time date and time display

## 💡 How It Works

- Enter quantity of items (Fries, Burger, Pizza, Drinks, etc.)
- Click TOTAL to calculate full bill
- View tax, service charge, and total amount
- Use RESET to clear all fields
- Use PRICE to check item prices
- Use calculator for quick calculations
